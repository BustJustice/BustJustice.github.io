发表于 2020-12-12 更新于 2020-12-17 评论数：

最初的方法

在介绍 class 继承以前，先来回忆一下没有 class 之前类是怎么被创建和继承的：

1、定义 function father 构造函数，再通过 prototype 定义 father 类原型方法。

2、定义 function child 构造函数，构造函数内部通过 father.call( this, arguments ) 的方式获得父类的实例属性。

3、通过 child.prototype = new father() 的方式获取 father 的原型方法。

4、最后还必须 child.prototype.constructor = child 的方式进行子类构造函数的还原。

function Person(name, age) {
    this.name = name
    this.age = age
  }
  Person.prototype.setName = function (name) {
    this.name = name
  }

  function Student(name, age, price) {
    Person.call(this, name, age) //得到父类型的属性
    this.price = price
  }
  Student.prototype = new Person()  //得到父类型的方法
  Student.prototype.constructor = Student
  Student.prototype.setPrice = function (price) {
    this.price = price
  }

this/call/apply/bind

发表于 2020-12-12 更新于 2020-12-16 评论数：

this指向

1.new调用绑定到新创建的对象
2.call或者apply调用指向绑定的第一个参数
3.对象的函数调用绑定到那个对象
4.普通函数调用严格模式下绑定到undefined，否则绑定全局对象
5.箭头函数会继承外层函数的this绑定，没有外层函数就绑定到全局对象
6.DOM一般指的是绑定事件DOM元素

通过new操作符条用构造函数

会经历一下四个阶段
1.创建一个新的对象
2.将构造函数的this指向这个新对象
3.指向构造函数的代码，对这个对象添加属性和方法
4.返回新对象

JavaScript深入之call和apply的模拟实现

所以我们模拟的步骤可以分为：

1.将函数设为对象的属性
2.执行该函数
3.删除该函数

作用域链的创建

函数的作用域在函数定义的时候就决定了
1.函数创建阶段，因为函数有一个内部属性[[scope]]，
在函数创建的时候，就会保存所有父变量对象到其中，
可以理解[[scope]]就是所有父变量的层级链，但是注意[[scope]]不代表完整的作用域链

2.函数激活，进入函数的上下文，创建VO/AO后，就会将活动对象添加到作用域链的前端

执行上下文的生命周期如下

创建{生成变量对象，简历作用域链，确定this指向}
执行{变量赋值，函数引用，执行其他代码}
执行完毕后出栈
